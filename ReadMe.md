Att göra
=========

1. Ladda ner koden och öppna den i Visual Studio.
2. Kör programmet.
3. Lägg till: Välkommen till Black Jack! ...
4. Spela flera gånger utan att starta om programmet.
5. Lägg till regler, se nedan.
6. Rita objektdiagram. Detta kan göras tidigare.


Regler
-----------


1. Player drar tills nöjd. 
2. Förlorar direkt om mer än 21.
3. Player vinner direkt om han har 21.
4. Dealer drar tills han har 17 eller mer.
5. Player vinner om han har högre än dealern.
6. Knekt, Dam och Kung räknas som 10. Ess räknas som 11.
7. Avancerat: Ess räknas som 1 istället för 11 om det behövs för att inte bli tjock (mer än 21).)
8. Extra: betting, splitta handen, ...
